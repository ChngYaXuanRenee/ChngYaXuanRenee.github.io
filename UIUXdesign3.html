Furnitopia Web Design

Programmed both front-end and back-end with API code. 3D furniture models were created in MAYA and textured in Substance Painter.

This e-commerce website provides a unique, immersive shopping experience, simulating 3D in-person purchasing. Products have three-dimensional images for detailed viewing. Gamification and interactive design enhance user engagement while offering branding opportunities for retailers and suppliers.
